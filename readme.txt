Embed a simple github widget for a repo

== Description ==

Provides a shortcode, [github repo="username/repo"], which creates an attractive widget in your page. 

Requires jquery (which is included with WordPress).

== Installation ==

1. Save this plugin in your wp-content/plugins directory
1. Activate the plugin through the 'Plugins' menu in WordPress
